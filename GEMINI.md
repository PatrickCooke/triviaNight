# TriviaNight: Local-First Event Management & Presentation

Senior Full-Stack Engineer-led project for a Raspberry Pi-hosted trivia system.

## Project Vision
A robust, local-first tool for managing trivia "Events", "Sets", and "Questions" with a slide-based presentation interface and performance-driven analytics.

## Core Mandates
- **Performance:** Optimized for Raspberry Pi hardware.
- **Reliability:** SQLite-backed local-first data integrity.
- **UX:** Seamless slide-based navigation and intuitive CRUD for data entry.
